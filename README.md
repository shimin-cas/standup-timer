# Standup Timer

A comprehensive web application for managing and timing team standup meetings with advanced performance tracking and reporting capabilities.

## 🚀 Live Demo

**[Try it now: https://shimin-cas.github.io/standup-timer/](https://shimin-cas.github.io/standup-timer/)**

## ✨ Features

### Meeting Management
- **Manual Timer Control**: Start, pause, resume, and advance speakers with explicit controls
- **Attendance Management**: Mark members absent with reasons before and during meetings
- **Speaker Sequencing**: Visual queue with click-to-advance functionality
- **Explicit Meeting End**: Clear "End Meeting" button for better control

### Team Management
- **Member Profiles**: Store name, email, and team assignments
- **CSV Import/Export**: Bulk operations for managing large teams (30+ members)
- **Team Organization**: Group members by teams for better analytics

### Template System
- **Reusable Configurations**: Create meeting templates with predefined speaker sequences
- **Drag-and-Drop Sequencing**: Easily reorder speakers with visual feedback
- **High-Density Interface**: View and manage 20-30+ members without scrolling

### Advanced Reporting
- **Comprehensive Analytics**: Team-level breakdowns with totals and averages
- **Performance Leaderboards**: Visual bar charts with rankings and animal emojis
- **Detailed Reports**: Individual speaking times, variance analysis, and absentee tracking
- **Export Capabilities**: CSV export for reports and team data

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/shimin-cas/standup-timer.git
   cd standup-timer
   ```

2. **Serve locally**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

## 📱 Usage

1. **Add Team Members**: Go to the Team tab and add your team members
2. **Create Templates**: Set up reusable meeting configurations with speaker sequences
3. **Start Meeting**: Select a template and configure attendance
4. **Run Standup**: Use timer controls to track each speaker's time
5. **Review Reports**: View detailed analytics and export data
